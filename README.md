# STAT405 Project: Aircraft Delay Propagation

This repository contains code and presentation assets for analyzing delay carry-over in U.S. domestic flights.

## Included

- `scripts/`: data cleaning, pairing, modeling, and reporting scripts
- `chtc/`: CHTC submit and helper files for parallel runs
- `results/report_figures/`: finalized figures and briefing documents (EN/ZH)

## Excluded

Large raw/processed data files are excluded from Git to keep the repository lightweight and portable.
